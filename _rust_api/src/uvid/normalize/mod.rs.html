1//! VCF variant normalization
2//! ([Tan et al. 2015](https://doi.org/10.1093/bioinformatics/btv112)).
3//!
4//! This module implements left-alignment and parsimonious trimming of
5//! VCF variant alleles so that differently-described variants that are
6//! biologically identical map to the same representation.
7//!
8//! # Design
9//!
10//! The module is intentionally free of UVID-specific types (`ChrIndex`,
11//! `Assembly`, etc.) and works entirely with generic primitives
12//! (`&str`, `u32`, `&[u8]`).  This makes it trivially extractable into
13//! a standalone crate.
14//!
15//! # Usage
16//!
17//! ```ignore
18//! use uvid::normalize::{normalize, InlineReference};
19//!
20//! let mut reference = InlineReference::new(Default::default())
21//!     .with_contig("chr1", b"ACGTACGTACGT");
22//!
23//! let result = normalize("chr1", 5, b"TACG", b"T", &mut reference).unwrap();
24//! assert!(result.was_modified);
25//! ```
