pub fn find_reference(assembly: &str) -> Result<PathBuf, ReferenceError>
Expand description

Search the data directory for a reference genome matching assembly.

Tries each extension in [EXTENSIONS] order, returning the first existing file. For example, for assembly "GRCh38" it checks:

  • <data_dir>/GRCh38.2bit
  • <data_dir>/GRCh38.fa
  • <data_dir>/GRCh38.fasta

Returns the path to the first match, or a ReferenceError::NotFound listing all paths that were tried.
